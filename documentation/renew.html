Renew

Expired callback renewal

This method does nothing if the callback has not expired or cancelled

Immediate

Call renew without arguments to immediately mark a callback as invokable (this is the default state of all callbacks). This will have no effect on future callback expirations, and can be used as many times as needed. 

new Callback()
.renew()

Event driven

Similar to cancel, renew uses when under the hood, allowing you to use every event tool when supports. As soon as your event fires, your expired callback is renewed for execution. 

new Callback( function )
.when( product, 'click' )
.cancel()
.renew( request, 'load' )

The above example creates a new callback, binds it to click events on the product, and prevents the callback from running until the request has come in successfully.
