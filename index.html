CallbackJS

Callbacks with super powers

Documentation

What it does for you

CallbackJS empowers your event-driven functions, giving you simplified decisions, error handling, event binding, load promises, cancellations, and custom event listeners.

Cancellations

You might want to cancel your callback after some point. The way this library handles that is by cancelling. Once a callback has been cancelled, you can enable it again by renewing. Each action can be event driven, allowing you to write powerful, succinct callback declarations.

Decisions

If you find yourself checking application state inside your callback, there's a better way. CallbackJS lets you set a conditional function, one that is run before your callback. If it returns falsey, your callback won't be invoked.

Custom Events

This is the reason CallbackJS was designed. Although JavaScript is event oriented, defining your own can be a painful experience, and they only apply to HTML elements.

This library gives you simple tools for declaring and firing your own custom events.
